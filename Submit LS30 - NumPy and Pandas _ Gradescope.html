Q2

2 Points

Whereas NumPy is powerful for scientific computations, for data analysis involving data tables, such as those explored in previous projects and exercises (such as CSV-formatted data), the pandas library is more appropriate.

Having a high-level understanding of numpy is important before using pandas, though, because pandas makes use of numpy in its columnar-data format.

A table in pandas is called a DataFrame. It is a column-oriented data structure like you have utilized and built by-hand earlier in the course.
